## 🚀 Getting Started

### Prerequisites
- Git installed on your machine
- GitHub account

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/parvezkhan/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (with http-server package)
   npx http-server
   ```

3. **View locally**
   - Navigate to `http://localhost:8000` in your browser

### Customization

1. **Update personal information**
   - Edit the content in `index.html` with your own information
   - Update social media links
   - Modify contact details

2. **Customize colors**
   - Edit CSS variables in `css/style.css`:
   ```css
   :root {
       --primary-color: #2563eb;
       --secondary-color: #1e40af;
       --accent-color: #10b981;
       /* ... more variables */
   }
   ```

3. **Add or modify sections**
   - Edit HTML structure in `index.html`
   - Add corresponding CSS in `css/style.css`

## 🌐 Deployment on GitHub Pages

### Method 1: Automated Deployment

1. **Create a GitHub repository**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Portfolio website"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
   git push -u origin main
   ```

2. **Enable GitHub Pages**
   - Go to repository settings
   - Scroll to "GitHub Pages" section
   - Select "main" branch as source
   - Click Save

3. **Access your portfolio**
   - Your site will be available at: `https://YOUR_USERNAME.github.io/portfolio`

### Method 2: Using GitHub Desktop

1. Open GitHub Desktop
2. File → Clone Repository → Select the portfolio repository
3. In repository settings, publish to GitHub
4. Enable GitHub Pages in repository settings
